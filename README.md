# I7_PerimetersFinal

Este repositorio permite calcular los perimetros de las figuras: 
- Triangulo 
- Rectangulo  
- Cuadrado 
- Pentagono 
- Hexagono 
- Circulo 
Para medidas de numeros naturales y de punto flotante.


## Estructura de directorios

La estructura que se realizo para llevar acabo la creacion de las bibliotecas estatica y dinamica, es la siguiente:
```
C:.
│   main.cc
│
├───.vscode
│       settings.json
│
├───app
│       test.exe
│       testD.exe
│
├───include
│       Perimeters
│
├───lib
│   ├───dinamic
│   │       perimetersD.dll
│   │
│   └───static
│           perimetersS.lib
│
├───obj
│       circulo.o
│       cuadrado.o
│       hexagono.o
│       pentagono.o
│       rectangulo.o
│       triangulo.o
│
└───src
        circulo.cc
        Cuadrado.cc
        Hexagono.cc
        Pentagono.cc
        rectangulo.cc
        triangulo.cpp
```
